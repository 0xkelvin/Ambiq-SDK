Detailed Description

Global type definitions for MSPI device functions.

the abstracted marco for the various mspi devices

Function pointer structure for MSPI device operations including initialization, read/write operations, and timing configuration

the universal struct for the various mspi devices struct each internal function pointer members refer to their different drivers
