SYSPLL - System PLL
mcu » apollo510

Detailed Description

Purpose: This module provides comprehensive functions for interfacing with the System PLL (Phase-Locked Loop) on Apollo5 devices. It supports PLL configuration, frequency generation, lock management, and clock multiplexing for precise clock generation and distribution.

Key Features

  1. PLL Configuration: Flexible PLL parameter configuration.
  2. Frequency Generation: Precise frequency generation and control.
  3. Lock Management: PLL lock detection and wait functionality.
  4. Clock Multiplexing: Support for multiple clock source selection.
  5. Bypass Mode: PLL bypass functionality for direct clock routing.

Functionality

  • Initialize and configure System PLL
  • Generate precise clock frequencies
  • Handle PLL lock detection and management
  • Support clock multiplexing and selection
  • Enable/disable PLL and bypass modes

Usage

  1. Initialize PLL using am_hal_syspll_initialize()
  2. Configure PLL parameters and frequencies
  3. Enable PLL and wait for lock
  4. Configure clock multiplexing as needed
  5. Disable PLL when no longer needed

Configuration

  • PLL Parameters: Configure reference and feedback dividers
  • Frequency Settings: Set up output frequencies and VCO ranges
  • Lock Timing: Configure lock detection and wait times
  • Clock Sources: Set up clock multiplexing and selection

Generate system PLL configuration to achieve desired VCO Frequency
This function generates the following configurations for system PLL. configuration not listed needs to be set in separately.

  • eFractionMode
  • ui8RefDiv
  • ui16FBDivInt
  • ui32FBDivFrac
Parameters

fVCOClkMHz- system PLL VCO output frequency desired in MHz
Returns

  • AM_HAL_STATUS_SUCCESS: configuration succesfully generated
  • AM_HAL_STATUS_INVALID_ARG: invalid arguments received.
  • AM_HAL_STATUS_OUT_OF_RANGE: fVCOClkMHz out of range.
  • AM_HAL_STATUS_FAIL: No valid configuration found.
