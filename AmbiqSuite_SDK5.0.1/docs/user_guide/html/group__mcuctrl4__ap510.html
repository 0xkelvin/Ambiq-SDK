MCUCTRL - MCU Control
mcu » apollo510

Detailed Description

Purpose: This module provides functions for interfacing with the MCU Control (MCUCTRL) module on Apollo5 devices. It supports device information retrieval, external clock control, status monitoring, and chip configuration for system management and initialization.

Key Features

  1. Device Information: Retrieve chip ID, revision, and vendor information.
  2. Memory Configuration: Access memory size and configuration data.
  3. External Clock: Manage external clock status and configuration.
  4. System Control: Provide system control and status operations.
  5. Feature Detection: Identify device features and capabilities.

Functionality

  • Retrieve device information and chip identification
  • Access memory configuration and size information
  • Manage external clock status and configuration
  • Provide system control and status operations
  • Support for device feature detection and configuration

Usage

  1. Get device information using am_hal_mcuctrl_info_get()
  2. Retrieve memory configuration as needed
  3. Check external clock status
  4. Access system control functions
  5. Query device features and capabilities

Configuration

  • Device Info: Access chip ID, revision, and vendor data
  • Memory Sizes: Retrieve ITCM, DTCM, and SSRAM configurations
  • External Clocks: Configure external clock sources
  • System Control: Set up system control operations
