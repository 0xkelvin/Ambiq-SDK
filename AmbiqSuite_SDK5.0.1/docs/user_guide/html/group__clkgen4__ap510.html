CLKGEN - Clock Generator
mcu » apollo510

Detailed Description

Purpose: This module provides functions for interfacing with the Clock Generator (CLKGEN) on Apollo5 devices. It supports clock source control, frequency adjustment, clock output configuration, and interrupt handling for flexible clock generation and management.

Key Features

  1. Clock Source Control: Manage various clock sources and frequencies.
  2. Frequency Adjustment: HFRC and HFRC2 frequency adjustment capabilities.
  3. Clock Output: Configurable clock output generation.
  4. Interrupt Support: Interrupt handling for clock events.
  5. Power Management: Clock power control and management.

Functionality

  • Control clock sources and frequency generation
  • Adjust HFRC and HFRC2 frequencies
  • Configure clock output parameters
  • Handle clock interrupts and status monitoring
  • Manage clock power states

Usage

  1. Control clock sources using am_hal_clkgen_control()
  2. Adjust frequencies with am_hal_clkgen_hfrcadj_target_calculate()
  3. Configure clock outputs as needed
  4. Handle clock interrupts and status
  5. Manage clock power states

Configuration

  • Clock Sources: Configure HFRC, HFRC2, and other clock sources
  • Frequency Adjustment: Set up frequency adjustment parameters
  • Clock Output: Configure clock output generation
  • Interrupts: Set up interrupt sources and handlers
